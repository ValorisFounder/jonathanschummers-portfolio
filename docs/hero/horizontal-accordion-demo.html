L'animation "expand/collapse" pour testimonials, galeries, ou showcase de profils

Survole chaque card pour voir l'expand + le texte

/* Chaque item est collapsed par défaut (80px) */ .item { flex: 0 0 80px; transition: flex 0.6s cubic-bezier(0.4, 0, 0.2, 1); } /* Le premier est ouvert par défaut */ .item:first-child { flex: 1; /* prend tout l'espace restant */ } /* Au hover du PARENT : tout collapse */ .accordion:hover .item { flex: 0 0 80px; } /* Au hover de L'ITEM : celui-là s'expand */ .accordion:hover .item:hover { flex: 1; } /* Le texte apparaît avec un délai */ .content { opacity: 0; transform: translateY(10px); transition: opacity 0.4s ease 0.15s, transform 0.4s ease 0.15s; } .accordion:hover .item:hover .content { opacity: 1; transform: translateY(0); }
Le trick CSS : Tu cibles .accordion:hover .item pour tout collapser, puis .accordion:hover .item:hover pour expand celui survolé. La propriété flex gère tout — pas besoin de width.

Clique sur chaque portrait — les images collapsed sont assombries/désaturées

Les détails qui font la différence :
— Images collapsed = filter: brightness(0.6) saturate(0.8) → désaturées et sombres
— Image active = brightness(0.85) saturate(1) → vivante
— Le texte a un transition-delay: 0.25s → il arrive APRÈS l'expansion (pas pendant)
— Le guillemet décoratif en ::before ajoute du polish
— Le fond dark du parent (#0f172a) unifie le tout

Comment intégrer dans ton portfolio

1. Remplace les gradients par de vraies photos de tes clients/collaborateurs (background-image: url(sarah.webp))
2. Utilise la Technique 3 (version dark + click JS) — c'est la plus portfolio-ready
3. Si tu n'as pas de photos de tes clients, utilise des avatars ou des initiales sur fond coloré
4. 5-6 testimonials max — au-delà les cards collapsed deviennent trop étroites
5. Lie chaque testimonial au case study correspondant via le tag en bas
